# 一些coding的规范

## 非必要不用global
- 尽可能通过参数传递
- 用函数来返回结果

## 限定好函数作用阈值
- 函数仅返回函数内的计算结果
- 不要直接修改外部变量，可变对象传进函数都是引用的，会被改掉

## 切分函数功能
- 尽可能让每个函数各司其职、职责明确
- 不要把大量代码放在一个函数单元
- 职责单一原则，分 single 和 multi

## 写好 Typing 注解
- 传入什么？
- 返回什么？用 | 来分情况

## 失败记录
- try + except + finally
- 同时用 logging 打印、记录必要信息
- logging 养成习惯，大项目时写道utils里面，进行封装
